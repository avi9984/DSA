What is javascript and how its work?
Ans:- javascript is a light-weight, intrepreted programing language. It is scripting language for web pages. javascript is single threaded language.

How its work?
javascript work on four steps
step1: parser: parse the code
step2: AST: if parser is satisfied there is no error then create data structure callled abstract syntax tree(AST).
step3: conveted to machine code
step4: machine code
